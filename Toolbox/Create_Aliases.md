# How to create aliases on iTerm

```~/.zshrc```

Open file and uncomment
```export PATH=$HOME/bin:/usr/local/bin:$PATH```

Then add the alias this way
```
alias ga="git add"
alias gc="git commit -m"
alias gp="git push"
alias gs="git status"
```
